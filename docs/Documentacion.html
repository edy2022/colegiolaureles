Requerimientos funcionales

La siguiente Especificación de Requerimientos de Software (SRS) del sistema a construir, surge con la finalidad de proveer toda la información de lo que quiere el estudiante o docente que haga el software. Tales requerimientos y requisitos son la base a la hora de comenzar el desarrollo del Software. En este caso el software a desarrollar es un software para el control de notas y de mas en el Colegio los Laureles.

Acta de Constitución del Proyecto

Descripción

El Acta de Constitución del Proyecto es un documento en el que se define el alcance, los objetivos y los participantes del proyecto. Da una visión preliminar de los roles y responsabilidades, de los objetivos, de los principales interesados. Sirve como referencia de autoridad para el futuro del proyecto.

Plan de Contratacion

1. Objetivo

Desarrollar el paso a paso para crear el software que servirá para registrar las notas de los estudiantes de los Laureles, haciendo más eficaz llevar el registro de los estudiantes. Modificar los datos registrados como buscar, actualizar y eliminar. Para tener mayor seguridad en los datos ingresados.
